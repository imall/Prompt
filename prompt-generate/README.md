# prompt-generate

模組化的 AI 繪圖 Prompt 產生器，純前端、可部署於 GitHub Pages。

把 Prompt 當成「物件」一樣自由組合：角色 / 服裝 / 身材 / 姿勢 / 場景 / 光影畫風 / 鏡頭 / 負面 / 母版，
即時預覽、一鍵複製，所有資料儲存於瀏覽器 `localStorage`，並支援 JSON 匯入匯出。
