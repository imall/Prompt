# 雨天都市街道

大雨天的日本住宅街，適用都市系列。

---

【場景 Environment】

乾淨整齊的日本都市住宅街。

街道設定：
街道安靜、整潔、有秩序，沒有擁擠人群、沒有雜亂招牌、沒有過多車輛、沒有堆積雜物。
道路兩側是簡潔的日式住宅、低矮圍牆、少量整齊路樹、少量路燈與遠方模糊的都市街景。
整體街景要有留白與空間感，不要資訊過載。

天氣：
白天的大雨天，天空被厚重雨雲覆蓋，整體天色偏昏暗灰藍。
空氣濕冷、帶有雨霧與水氣。
柏油路面被大雨完全打濕，形成清楚的濕亮質感、積水、水花與柔和倒影。

氛圍：
清澈的雨幕、乾淨的街景、潮濕的空氣感、路面反光，以及角色撐著透明雨傘在雨中安靜行走的電影感。

【構圖特別說明（搭配 16:9 母版）】

鏡頭位置約在人物腰部到胸口高度之間，像站在雨中街道左側偏前方拍攝。
角色位於畫面左側三分之一處，靠近前景，人物約佔畫面高度 55% 到 70%。
街道透視明確往「畫面右後方」延伸，消失點位於畫面右半部遠處。
前景：角色與透明雨傘；中景：濕亮柏油路面；遠景：往右後方延伸的住宅街。

避免：雜亂街景、過多招牌、過多電線、擁擠人群、霓虹燈過多、廣告牌過多、建築過度密集。
