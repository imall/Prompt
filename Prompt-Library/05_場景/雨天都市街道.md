# 雨天都市街道

---

【場景 Environment】

乾淨整齊的日本都市住宅街，大雨天氣。

街道設定：
街道安靜、整潔、有秩序。
道路兩側是簡潔的日式住宅、低矮圍牆、少量整齊路樹、少量路燈與遠方模糊都市街景。
整體街景要有留白與空間感，不要資訊過載。

天氣：
天空被厚重雨雲覆蓋，整體天色偏昏暗灰藍。
柏油路面被大雨打濕，形成濕亮質感、積水與水花。

空間氛圍：
清澈雨幕、乾淨街景、潮濕空氣感、電影感。

避免：雜亂街景、過多招牌、過多電線、擁擠人群、霓虹燈過多、建築過度密集。
