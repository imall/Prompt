# 日本教室（靠窗座位）

午後陽光灑落的校園教室，適用校園系列。

---

【場景 Environment】

日本教室，午後陽光，靠窗座位。

場景設定：
柔和逆光、白色窗簾被微風吹起、木質課桌椅反射溫暖光線。
窗外有明亮藍天與綠意，空氣中有細微塵埃與光粒。
整體帶有安靜、清澈、青春感與淡淡懷舊感。

光影：
光線從左側窗戶照入，形成柔和輪廓光、髮絲高光、衣物邊緣光與溫暖高光。
午後陽光灑在木質課桌、窗框、窗簾與人物肩膀、頭髮、臉頰上。
窗外綠意的微弱反射、木質桌椅的暖色反射、窗簾透光的柔白亮度。
前景桌面、角色椅背與人物之間有自然遮擋關係與接觸陰影。

【構圖特別說明（搭配 16:9 母版）】

視角：「坐在她後方座位的人視角」。
鏡頭位於她後方稍偏右，高度接近後座坐姿眼睛高度。
前景下方有自己的木質課桌橫向擋住畫面下緣，形成隔桌看前座的視角感。
自己的桌面遮住女主角大部分大腿，但桌緣以下或兩側仍可見一部分大腿與裙擺。

女主角位於畫面中央偏左，靠窗位置。
遠景：教室空間往後方延伸，其他課桌椅、黑板或窗戶在右側背景柔化呈現。
