# 摩天輪夜景

---

【場景 Environment】

夜晚巨大摩天輪車廂內。

車廂設定：
車廂位於高空，窗外可見都市夜景、河岸燈海、遠方高樓、深藍夜空、薄雲與月光。
玻璃窗反射城市霓虹與車廂內部。
車廂內有柔和暖色燈光，窗外為冷色夜景。

背景要求：
背景夜景需要非常美，但不能搶走人物主體。
呈現都市燈海、河岸反射光、遠方高樓群、柔和雲層、摩天輪外部燈光與玻璃窗細微反射。
背景細節要有深度與距離感，遠景柔和景深，近景窗框與車廂內部高級質感。

空間氛圍：
浪漫、夢幻、夜晚約會感、電影感。
