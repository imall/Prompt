# 摩天輪夜景

夜晚摩天輪車廂場景，適用紫髮少女約會系列。

---

【場景 Environment】

夜晚巨大摩天輪車廂內。

車廂設定：
車廂位於高空，窗外可見東京系都市夜景、河岸燈海、遠方高樓、深藍夜空、薄雲、星光與月光。
玻璃窗反射城市霓虹、車廂暖光與角色長髮。
車廂內是柔和暖色燈光，與窗外冷色夜景形成浪漫對比。
空氣中有細微光粒、星塵感與夢幻散景。

背景要求：
背景夜景需要非常美，但不能搶走人物主體。
呈現東京系都市燈海、河岸反射光、遠方高樓群、星空、柔和雲層、摩天輪外部燈光、玻璃窗上的細微反射。
背景細節要有深度與距離感，遠景有柔和景深，近景窗框與車廂內部有高級質感，但不要太暗。

【構圖特別說明（搭配 16:9 母版）】

人物位於畫面中間偏右，靠近右側三分之一位置。
角色約佔畫面高度 75%–85%。
鏡頭是近距離戀愛視角，中低腰部視角，像坐在她正前方的位置。
左側與後方保留摩天輪窗景、城市燈海、夜空與柔和散景。
