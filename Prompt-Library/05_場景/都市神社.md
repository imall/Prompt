# 都市神社（大型廣場）

大型都市神社寬闊境內廣場，適用巫女服系列。

---

【場景 Environment】

大型都市神社的寬闊境內廣場。

整體空間開闊、明亮、有尺度感，不是狹小山林神社，也不是小庭院神社。

空間層次：
- 前景：角色與神樂舞動作
- 中景：大片空曠石板廣場、柔和樹影與少量石燈籠
- 背景：遠處的大型木造拜殿（規模宏大，有寬大的屋頂、粗壯木柱、朱紅與深木色結構）
- 遠景：朱紅色鳥居、樹冠與明亮天空（鳥居放在更遠的廣場入口或畫面側後方）

時間與光線：
清晨或午後，陽光從側後方照入，形成柔和輪廓光、髮絲高光、衣物邊緣光、光斑與空氣中漂浮的細微塵埃。

環境光影響：
朱紅色鳥居與木造建築的暖色反射光輕微映在白色服裝與皮膚暗部；樹影在衣袖與裙上形成柔和斑駁陰影；角色腳下有自然接觸陰影。
