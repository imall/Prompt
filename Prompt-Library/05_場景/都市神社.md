# 都市神社（大型廣場）

---

【場景 Environment】

大型都市神社的寬闊境內廣場。

整體空間開闊、明亮、有尺度感，不是狹小山林神社，也不是小庭院神社。

空間層次：
- 前景：角色與動作區域
- 中景：大片空曠石板廣場、少量石燈籠
- 背景：大型木造拜殿（寬大屋頂、粗壯木柱、朱紅與深木色結構）
- 遠景：朱紅色鳥居、樹冠與明亮天空

空間氛圍：
開闊、莊嚴、清澈、傳統日本神社感。
