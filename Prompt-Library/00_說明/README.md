# Prompt Library

AI 圖片生成 Prompt 模組化管理系統。

## 目錄結構

| 資料夾 | 說明 |
|--------|------|
| `00_說明/` | 使用說明與命名規則 |
| `01_母版/` | 完整框架，定義比例、構圖、品質基準 |
| `02_角色/` | 角色身份、外觀、氣質模組 |
| `03_服裝/` | 服裝設定模組 |
| `04_姿勢/` | 動作與姿勢模組 |
| `05_場景/` | 場景與環境模組 |
| `06_光影畫風/` | 燈光、光影、整體畫風模組 |
| `07_負面Prompt/` | 負面 Prompt 防崩壞模組 |
| `08_成品Prompt/` | 組裝後的完整生成 Prompt |

## 組裝邏輯

```
母版（比例 + 畫風基準）
+ 角色模組
+ 服裝模組
+ 姿勢模組
+ 場景模組
+ 光影畫風模組
+ 負面 Prompt
```

母版就像 abstract class，定義框架；模組是可抽換的零件。
`08_成品Prompt/` 下的檔案是組裝後的完整版本，直接複製使用。
