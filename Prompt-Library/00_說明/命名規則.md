# 命名規則

## 模組檔案

`[描述性名稱].md`

簡短直覺，例如：`黑長直學姐.md`、`MUJI日式公寓.md`、`柔和晨光.md`

## 成品 Prompt

`日期_角色_場景_服裝_動作_比例.md`

範例：
- `2026-05-22_黑長直學姐_MUJI客廳_CK內衣_慵懶側躺_9x20.md`
- `2026-05-18_紫髮少女_摩天輪夜景_約會洋裝_坐姿_16x9.md`
- `2026-05-20_紫髮少女_深夜臥室_吊帶睡衣_熟睡_16x9.md`

若動作不重要可省略：
- `2026-05-15_紫髮少女_都市神社_巫女服_16x9.md`

## 比例縮寫

| 縮寫 | 說明 |
|------|------|
| `9x20` | 9:20 直式手機桌布 |
| `16x9` | 16:9 橫幅 |
