# 白色水手服制服

適用學校場景的日系校園制服。

---

【服裝 Outfit】

白色水手服風格制服。

服裝組成：
- 白色短袖上衣（上衣無口袋）
- 黑色百褶裙
- 黑色領結或細緞帶裝飾

布料質感：
布料較薄，但整體端莊、自然、完整覆蓋。

整體服裝必須保持乾淨、端莊、自然、電影感，不低俗、不刻意性感化。

在雨天場景中：
雨傘無法完全遮擋的肩膀部位因雨水略微打濕，呈現自然的濕潤感；黑色裙子下擺亦被雨水浸濕，呈現真實布料吸水後的垂墜感。
