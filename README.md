# Prompt

AI 繪圖 Prompt 模組化管理系統，含素材庫與產生器工具。

## 專案結構

```
Prompt/
├── Prompt-Library/        # Prompt 模組素材庫
│   ├── 00_說明/           # 使用說明與命名規則
│   ├── 01_母版/           # 比例與畫風基準框架
│   ├── 02_角色/           # 角色外觀模組
│   ├── 03_服裝/           # 服裝設定模組
│   ├── 04_姿勢/           # 動作與姿勢模組
│   ├── 05_場景/           # 場景與環境模組
│   ├── 06_光影畫風/       # 燈光、光影、畫風模組
│   ├── 07_負面Prompt/     # 負面 Prompt 防崩壞模組
│   └── 08_成品Prompt/     # 組裝後的完整可用 Prompt
└── prompt-generate/       # 模組化 Prompt 產生器（規劃中）
    └── docs/              # 設計文件
```

## Prompt-Library

以「模組化」概念管理 AI 繪圖 Prompt。每個模組是獨立的 `.md` 檔案，組裝邏輯如下：

```
母版（比例 + 畫風基準）
+ 角色模組
+ 服裝模組
+ 姿勢模組
+ 場景模組
+ 光影畫風模組
+ 負面 Prompt
```

`08_成品Prompt/` 下存放已組裝完成的完整 Prompt，依角色分資料夾，直接複製使用。

### 命名規則

- **模組**：`[描述性名稱].md`（例：`黑長直學姐.md`、`MUJI日式公寓.md`）
- **成品**：`日期_角色_場景_服裝_動作_比例.md`（例：`2026-05-22_紫髮少女_日本教室_校服_整理頭髮_16x9.md`）

### 現有角色

| 角色 | 特徵 |
|------|------|
| 黑長直學姐（詩羽） | 黑長直髮、白色髮箍、紅棕色玻璃感眼睛 |
| 紫髮少女（費倫） | 極長紫色長髮、整齊厚瀏海、紫色玻璃感眼睛 |

## prompt-generate

模組化 Prompt 產生器，純前端 Web App，計畫部署於 GitHub Pages。

將 Prompt 模組以 UI 介面自由組合，即時預覽、一鍵複製，資料存於 `localStorage` 並支援 JSON 匯入匯出。

- 技術：HTML + Vanilla JS（ES Modules）
- 狀態：🚧 規劃中
- 設計文件：[prompt-generate/docs/可行性文件.md](prompt-generate/docs/可行性文件.md)
